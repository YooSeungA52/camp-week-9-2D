# camp-week-9-2D

## 1. 2D 클리커 게임 구현 사항
1) 클릭 이벤트 처리
2) 자동 클릭 기능
3) 점수 시스템
4) 아이템 및 업그레이드 시스템
5) 사운드 이펙트
6) 애니메이션

## 2. 개선해야 할 점
- 봇들을 업그레이드 하면 시간은 감소하지만 클릭 보상이 증가하지 않음
- 봇들이 자동 클릭을 하면 Clicker의 애니메이션이 작동하지 않음
- 설정창에 볼륨 슬라이더밖에 없음 → 게임 초기화 등 기능 추가하기
- 저장 기능이 없음
- 코드 정리

## 3. 추가하고 싶은 점
- 업적 시스템을 만들고 싶었음(클릭한 횟수, 업그레이드 레벨 등)
- 상점을 더욱 다양한 아이템으로 꾸미고 싶음
